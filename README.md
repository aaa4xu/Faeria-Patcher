# Faeria Patcher
Faeria Patcher is util for debug Faeria network traffic

## Features
* Change game server IP
* Disable traffic encryption

## Using
`dotnet FaeriaPatcher.dll GameDirectory ServerIP`
* `GameDirectory` - Folder with `Faeria.exe`
